第一步：你做

创建飞书应用

  1. 访问 open.feishu.cn/app，创建 N 个企业自建应用
  2. 每个应用 → 应用功能 → 机器人 → 开启
  3. 每个应用 → 版本管理与发布 → 创建版本 → 发布
  4. 收集每个应用的 App ID、App Secret
  5. 新建群聊，把所有机器人都拉进去，获取群 chat_id
未发布版本会导致 @消息不响应，务必发布

扩展更多角色

只需在凭证里加上新角色即可，支持 2~5 个任意组合：

ROLES=pm,plan,dev,qa,test  # 加 QA
ROLES=pm,arch,dev,test       # 加架构师

新角色只需提供 SOUL 模板文件（templates/SOUL_{role}.md），setup.sh 自动识别并渲染。
